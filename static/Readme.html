Welcome to market web, the web interface for the Matching Engine that supports two matching algorithms viz., FIFO and Prorata.
For more information on these algorithms , please visit CFI institute web site

Market web is a stand alone web server that you can start on your desktop/laptop that runs Windows 10 or higher. YOu will need to install cargo setup to run the application using cargo. Alternately, you can download the binary(market-web.exe) from the app's repository
